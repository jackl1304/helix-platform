## 🚀 Neue Screenshots erstellen

### Option 1: Automatisch via Browser-Tools

```javascript
// Im Browser-Console ausführen:
await page.screenshot({
  type: 'png',
  path: './screenshots/demo-05-new-feature.png',
  fullPage: true
});
```

### Option 2: Manuell

1. Navigiere zu gewünschter Seite
2. Drücke `F12` (DevTools öffnen)
3. Cmd/Ctrl + Shift + P → "Screenshot"
4. Wähle "Capture full size screenshot"
5. Speichere im `screenshots/` Ordner

### Option 3: Tools verwenden

- **Windows:** Snipping Tool, ShareX
- **Mac:** Cmd + Shift + 4
- **Linux:** GNOME Screenshot, Flameshot
- **Browser Extensions:** Fireshot, Awesome Screenshot
